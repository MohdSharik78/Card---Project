🎴 React User Card Project

A simple and responsive React application that displays user information using reusable card components. This project demonstrates React component structure, props usage, and CSS styling.

🚀 Features

- Reusable Card Component
- User Information Display
- Clean UI Design
- Component-Based Architecture
- Easy to Customize and Extend

📁 Project Structure

src/
├── App.jsx
├── App.css
├── Card.jsx
├── User.jsx
└── main.jsx

Components

- App.jsx – Main application component.
- Card.jsx – Reusable card layout component.
- User.jsx – Displays user details inside cards.
- App.css – Application styling.
- main.jsx – React application entry point.

🛠️ Technologies Used

- React.js
- JavaScript (JSX)
- CSS3
- Vite

📦 Installation

Clone the repository:

git clone <repository-url>

Install dependencies:

npm install

Run the project:

npm run dev

🎯 Learning Objectives

This project was created to practice:

- React Components
- JSX Syntax
- Props
- CSS Styling
- Project Structure in React
